# California-Housing-Price-Prediction
A deep learning project that uses LSTM and GRU models to forecast California housing prices. The project includes data preprocessing, sequence generation, model training, hyperparameter tuning, and performance evaluation using PyTorch.


# California Housing Price Prediction

## Project Overview
This project focuses on predicting California housing prices using deep learning models. LSTM and GRU architectures were implemented to analyse sequential patterns and forecast housing prices.

## Dataset
The project uses the California Housing dataset from Scikit-Learn, containing information about housing characteristics and median house values.

## Project Workflow
- Data preprocessing and normalisation
- Sequence generation for sequential modelling
- LSTM model implementation
- GRU model implementation
- Hyperparameter tuning
- Model training and evaluation

## Technologies Used
- Python
- PyTorch
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Jupyter Notebook

## Models Used
- Long Short-Term Memory (LSTM)
- Gated Recurrent Unit (GRU)

## Results
The models were evaluated using Mean Squared Error (MSE) to compare prediction performance across different hyperparameter configurations.
